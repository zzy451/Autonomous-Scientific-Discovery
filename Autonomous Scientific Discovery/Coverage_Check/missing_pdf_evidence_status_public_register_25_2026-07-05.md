## Purpose

This public register is a publication-formatted derivative of the local progress owner file used in the Autonomous Scientific Discovery literature project.

It lists the confirmed-core papers that do **not** currently have a locally archived main-paper PDF, while preserving the current public evidence route used for indexing and structured management. Typical evidence routes include DOI landing pages, publisher HTML, official article pages, abstracts, supplementary files, project pages, and preprint routes.

This document is intended for GitHub publication and external inspection only. The local authoritative working file remains the internal progress owner file and is not modified by this public register.
