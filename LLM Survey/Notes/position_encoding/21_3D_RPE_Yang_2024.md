# 3D-RPE: Enhancing Long-Context Modeling Through 3D Rotary Position Encoding
**作者**: Zhijie Yang, Yiming Yang 等
**年份**: 2024 | **arxiv**: 2406.09897

## 0. 摘要翻译
受 Bloch 球表示的启发，3D-RPE 提出在三维球面上进行旋转位置编码，是标准 2D RoPE 的高维推广。3D-RPE 具有两大优势：可控的长程衰减和更高的位置分辨率，从而显著增强长上下文建模能力。

## 1. 方法动机
- 标准 RoPE 在二维平面（复平面）上进行旋转，每个频率维度对应一个 2D 旋转
- 2D 旋转的表达能力有限：衰减模式固定，高频维度在长距离时产生近似随机的干扰
- 量子力学中的 Bloch 球提供了一种在三维球面上表示状态的优雅框架
- 将旋转从 2D 扩展到 3D 可以引入额外的自由度来控制衰减行为

## 2. 核心方法

### 2.1 从 2D 到 3D 旋转
**标准 RoPE (2D)**：对每对维度 $(d_{2i}, d_{2i+1})$ 施加 2D 旋转矩阵
$$R_{2D}(\theta) = \begin{pmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{pmatrix}$$

**3D-RPE**：对每三个维度 $(d_{3i}, d_{3i+1}, d_{3i+2})$ 施加 3D 旋转矩阵
$$R_{3D}(\alpha, \beta, \gamma) = R_z(\alpha) R_y(\beta) R_z(\gamma)$$
使用 ZYZ 欧拉角参数化，其中 $\alpha, \beta, \gamma$ 均为位置的函数。

### 2.2 可控长程衰减
- 在 2D RoPE 中，注意力分数的衰减完全由频率 $\theta_i$ 决定，缺乏灵活性
- 3D-RPE 通过额外的旋转轴引入可调节的衰减包络
- 极角 $\beta$ 控制衰减速率：$\beta$ 越大，长距离衰减越快
- 可以为不同注意力头设置不同的衰减模式

### 2.3 位置分辨率提升
- 2D 旋转在高频时周期短，容易产生位置混叠
- 3D 旋转提供了更多的自由度来区分不同位置
- 在相同维度预算下，3D-RPE 的位置分辨率更高
- 减少了长序列中的位置碰撞概率

## 3. 实验结果
- 在 LLaMA 架构上验证（1.3B 和 7B 参数）
- 长上下文 perplexity 显著优于标准 RoPE
- 在 passkey retrieval 任务上扩展能力更强
- 在 LongBench 等下游任务上性能提升
- 额外计算开销极小（3D 旋转矩阵乘法仅增加约 50% 的 PE 计算量）

## 4. 与 2D RoPE 的对比
| 特性 | 2D RoPE | 3D-RPE |
|------|---------|--------|
| 旋转空间 | SO(2) 复平面 | SO(3) 球面 |
| 每组维度 | 2 维 | 3 维 |
| 衰减控制 | 仅频率 | 频率 + 极角 |
| 位置分辨率 | 较低 | 较高 |
| 参数 | $\theta_i$ | $\alpha_i, \beta_i, \gamma_i$ |

## 5. 与本综述的关联
- 代表了 RoPE 向高维旋转群推广的重要方向
- 与 LieRE（Lie 群推广）思路互补：3D-RPE 专注于 SO(3)，LieRE 探索一般 Lie 群
- 与 M-RoPE（多模态分解）不同：3D-RPE 是在同一模态内提升表达能力
- Bloch 球的物理直觉为位置编码设计提供了新视角
