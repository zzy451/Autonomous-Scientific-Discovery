# CLEX: Continuous Length Extrapolation for Large Language Models
**作者**: Guanzheng Chen, Xin Li, Zaiqiao Meng, Shangsong Liang, Lidong Bing | **年份**: 2023 (ICLR 2024) | **arxiv**: 2310.16450

## 0. 摘要翻译
CLEX 提出了一种基于常微分方程（ODE）的 RoPE 长度外推方法。不同于现有的离散缩放方法（如 PI、NTK-aware），CLEX 将位置编码的缩放因子建模为关于长度缩放比的连续动态系统，使用神经 ODE 学习不同频率维度的缩放因子如何随目标长度连续变化。实验表明 CLEX 可以将上下文窗口有效扩展到训练长度的 4 倍甚至 8 倍，且性能无明显下降。

## 1. 方法动机
- RoPE 的长度外推问题：模型在短序列上训练，需要在长序列上推理
- 现有方法的局限：
  - **Position Interpolation (PI)**: 线性压缩位置索引，对所有频率维度使用相同的缩放因子，过度压缩高频信息
  - **NTK-aware**: 对不同频率使用不同缩放，但缩放因子是手工设计的固定函数
  - **YaRN**: 改进了 NTK-aware，但仍是离散的、预定义的缩放策略
- CLEX 的洞察：最优的缩放因子应该是关于目标长度的连续函数，且不同频率维度的最优缩放策略不同——这可以用 ODE 来建模

## 2. 方法设计（重点：与六维度框架的关联）

### 2.1 将 PE 缩放建模为 ODE
- **标准 RoPE**: `θ_i = base^{-2i/d}`，位置 m 的旋转角度为 `m · θ_i`
- **缩放后的 RoPE**: `θ'_i(t) = f_i(t) · θ_i`，其中 t 是长度缩放比（target_len / train_len），f_i(t) 是第 i 个频率维度的缩放因子
- **CLEX 的 ODE 建模**:
  - 将 f_i(t) 的变化建模为 ODE: `df_i/dt = g_φ(f_i, t, i)`
  - g_φ 是一个小型神经网络（参数量极小），输入为当前缩放因子 f_i、缩放比 t、频率维度索引 i
  - 初始条件: f_i(1) = 1（缩放比为 1 时，即原始训练长度，不做缩放）
  - 使用 ODE solver（如 Euler 或 RK4）求解任意目标长度的缩放因子

### 2.2 训练策略
- 冻结原始 LLM 参数，只训练 ODE 网络 g_φ（参数量 < 0.5M）
- 在多个不同长度的数据上训练，让 ODE 学习缩放因子随长度变化的连续动态
- 训练目标：在扩展后的长度上最小化语言建模损失

### 2.3 连续性的优势
- **任意长度泛化**: ODE 定义了连续的缩放轨迹，可以外推到训练时未见过的长度
- **频率感知**: 不同频率维度有不同的缩放动态，由 ODE 自动学习
- **平滑过渡**: 缩放因子随长度平滑变化，避免离散方法的突变

### 与六维度框架的关联
- **位置编码维度**: CLEX 是 RoPE 长度外推方法的重要进展，属于位置编码的动态适应
- **与 PI/NTK/YaRN 的关系**: 可以视为这些方法的连续化推广——PI 是常数缩放，NTK 是固定函数缩放，CLEX 是学习的连续动态缩放
- **参数效率**: ODE 网络极小（<0.5M 参数），几乎不增加模型大小

## 3. 与其他方法对比
| 方法 | 缩放策略 | 频率感知 | 连续性 | 额外参数 | 最大外推倍数 |
|------|---------|---------|--------|---------|------------|
| PI | 线性压缩 | 否（统一） | 否 | 0 | ~2x |
| NTK-aware | 固定函数 | 是（手工） | 否 | 0 | ~4x |
| YaRN | 分段函数 | 是（手工） | 否 | 0 | ~4x |
| Code Llama | 频率缩放 | 是（手工） | 否 | 0 | ~4x |
| CLEX | 神经 ODE | 是（学习） | 是 | <0.5M | ~8x |
| iRoPE (Llama 4) | 架构级 | N/A | N/A | 架构改变 | ~40x |

- CLEX 在不改变模型架构的前提下实现了最大的外推倍数
- 相比 iRoPE 等架构级方案，CLEX 是即插即用的后训练方法

## 4. 实验表现
- 在 LLaMA-2 7B/13B 上验证，训练长度 4K，有效外推到 16K-32K
- 在长文本困惑度（perplexity）上显著优于 PI 和 NTK-aware
- 在 LongBench 等长上下文基准上表现优异
- 外推到 4x 训练长度时性能几乎无损，8x 时仍保持可用
- ODE 训练成本极低（几百步即可收敛）

## 5. 总结
a) CLEX 用神经 ODE 建模 RoPE 缩放因子随目标长度的连续动态变化，以极小的额外参数（<0.5M）实现了训练长度 4-8 倍的有效外推，是 RoPE 长度外推方法从离散到连续的范式转变。

b) 速记 pipeline: 目标长度 t → ODE solver: df/dt = g_φ(f, t, i) → 得到每个频率维度的缩放因子 f_i(t) → θ'_i = f_i(t)·θ_i → 缩放后的 RoPE 应用于注意力
