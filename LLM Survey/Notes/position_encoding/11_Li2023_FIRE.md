# FIRE: Functional Interpolation for Relative Positions
**论文**: Functional Interpolation for Relative Positions Improves Long Context Transformers
**作者**: Shanda Li, Chong You, Guru Guruganesh, Joshua Ainslie, Santiago Ontanon, Manzil Zaheer, Sumit Sanghai, Yao Yang, Sham Kakade, Srinadh Bhojanapalli
**年份**: 2023
**会议**: ICLR 2024
**arXiv**: 2310.04418

## 核心思想
FIRE 使用一个可学习函数（而非固定公式）将相对位置映射到 attention bias，并通过 **渐进式插值 (progressive interpolation)** 将任意长度输入映射到有界域，从而在不同序列长度上实现泛化。

## 方法细节

### 可学习位置函数
$$b_{ij} = f_\theta(\text{normalized}(|i-j|))$$

- f_θ: 可学习函数（用小型 MLP 实现）
- 输入为归一化后的相对位置
- 输出为标量 bias，加到 attention logit 上

### 渐进式插值 (Progressive Interpolation)
核心挑战：如何处理训练时未见的大相对距离？

FIRE 的方案：
1. 将相对位置映射到 [0, 1] 的有界区间
2. 使用分段线性或对数变换进行归一化
3. 无论输入序列多长，归一化后的输入始终在 [0, 1] 范围内
4. 可学习函数只需在 [0, 1] 上学习合适的映射

### 理论通用性
FIRE 可以表示（是其泛化）：
- **T5 RPE**: 对数分桶 + 可学习偏置 → FIRE 的特例
- **ALiBi**: 线性衰减偏置 → FIRE 的特例
- **Kerple**: 核函数偏置 → FIRE 的特例

这意味着 FIRE 理论上至少不会比这些方法差。

## Position Encoding 维度分析

### 分类
- **类型**: Attention Matrix Bias（可学习函数框架）
- **注入方式**: 在 attention logits 上加可学习偏置
- **参数量**: 一个小型 MLP 的参数量（per head 或共享）
- **长度外推**: 通过渐进式插值实现强泛化

### 关键创新
1. **函数化**: 从离散映射/固定公式 → 可学习连续函数
2. **归一化到有界域**: 解决 OOD 位置的根本问题
3. **理论统一**: 证明可表示多种现有方法

### 与其他方法的对比
| 方法 | 位置→偏置映射 | 是否可学习 | 有界输入 |
|------|-------------|-----------|---------|
| ALiBi | 线性 | 否 | 否 |
| T5 RPE | 分桶+查表 | 部分 | 是（通过分桶） |
| Kerple | 核函数 | 参数可学习 | 否 |
| FIRE | MLP | 完全可学习 | 是 |

## 实验结果
- 语言建模: 在 zero-shot 和 fine-tuning 场景下均显著提升长上下文泛化
- 训练长度 512/1024，测试 8k/16k+ 时保持低困惑度
- 优于 ALiBi、T5 RPE、Kerple

## 影响
- 提供了 attention bias 方法的最强统一框架
- 证明可学习函数优于手工设计的固定函数
- 渐进式插值技术有广泛适用性
