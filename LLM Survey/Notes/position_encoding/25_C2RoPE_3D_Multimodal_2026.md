# C²RoPE: Causal Continuous Rotary Positional Encoding for 3D Multimodal Reasoning
**作者**: (多机构合作)
**年份**: 2026 | **arxiv**: 2602.10551

## 0. 摘要翻译
C²RoPE 针对 3D 大型多模态模型（LMM）中 RoPE 的局限性，提出了一种改进方案，显式建模视觉处理中的局部空间连续性（Continuity）和空间因果关系（Causal）。标准 1D RoPE 在多模态处理中将视觉 token 展平为一维序列，破坏了视觉特征的空间连续性；C²RoPE 通过时空连续位置索引和因果空间关系建模来解决这一问题。

## 1. 方法动机
- 当前 3D 多模态模型（如处理点云、3D 场景的 LMM）直接继承了语言模型的 1D RoPE
- 1D 位置索引将 2D/3D 视觉 token 展平，破坏了空间邻近关系
- 例如：图像中相邻的两个 patch 在展平后可能获得相距很远的位置索引
- 视频帧间的时间连续性也被离散的帧索引打断
- 3D 推理任务（空间关系理解、深度估计等）需要精确的空间位置感知

## 2. 核心方法

### 2.1 空间连续性建模 (Continuity)
- 为视觉 token 分配连续的空间坐标而非离散的序列索引
- 图像 patch 使用 2D 连续坐标 $(x, y)$
- 视频 patch 使用 3D 连续坐标 $(x, y, t)$
- 坐标值可以是非整数，保持空间的连续性
- 相邻 patch 获得相近的坐标值，保持空间邻近关系

### 2.2 空间因果关系建模 (Causal)
- 在视觉 token 之间引入空间因果结构
- 定义空间扫描顺序（如光栅扫描、螺旋扫描）来建立因果方向
- 因果关系确保注意力计算中的位置偏置与空间结构一致
- 避免了展平操作引入的虚假位置关系

### 2.3 时空 RoPE 构造
将 RoPE 的频率维度分配给不同的时空轴：
$$R(x, y, t) = R_x(x) \otimes R_y(y) \otimes R_t(t)$$
- 每个轴使用独立的 RoPE 频率集
- 连续坐标允许亚像素级的位置编码
- 与 M-RoPE 类似的分解策略，但强调连续性和因果性

## 3. 实验结果
- 在 3D 视觉问答、空间关系推理任务上显著优于标准 1D RoPE
- 在 ScanQA、3D-LLM 等 3D 理解基准上提升明显
- 视频理解任务中时间连续性建模带来一致性改善
- 对深度估计和空间布局理解任务帮助最大

## 4. 与相关工作的对比
| 方法 | 空间连续性 | 因果建模 | 适用维度 | 目标场景 |
|------|:---:|:---:|:---:|------|
| 1D RoPE | 无 | 序列因果 | 1D | 文本 |
| M-RoPE | 离散 | 无 | 1D/2D/3D | 多模态 |
| 3D-RPE | 无 | 无 | 3D | 长上下文 |
| **C²RoPE** | **连续** | **空间因果** | **2D/3D** | **3D 多模态** |

## 5. 与本综述的关联
- 代表了 RoPE 在 3D 多模态推理中的最新应用
- 连续位置编码思想与 CLEX（连续长度外推）在不同维度上呼应
- 空间因果性是一个新颖的设计维度，超越了传统的序列因果
- 与 M-RoPE、3D-RPE 共同构成多维 RoPE 扩展的完整图景
