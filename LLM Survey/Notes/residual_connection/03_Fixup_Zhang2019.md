# Fixup Initialization: Residual Learning Without Normalization

**论文信息**: Zhang, H., Dauphin, Y.N., Ma, T. (2019)
**arXiv**: 1901.09321 | **会议**: ICLR 2019
**分类**: Control (残差初始化)

## 核心思想
Fixup (Fixed-update initialization) 通过精心设计的初始化方案，使残差网络无需归一化层即可稳定训练。

## 数学公式/关键组件
1. **残差分支最后一层权重初始化为 0**: 确保初始时为恒等映射
2. **权重缩放因子**: L^{-1/(2m-2)}，其中 L 为残差块数，m 为每块权重层数
3. **可学习标量乘子**: 每个残差分支一个，初始化为 1
4. **可学习标量偏置**: 每层之前，初始化为 0

## 实验结果
- 可训练深达 **10,000 层**的残差网络
- 在图像分类和机器翻译任务上达到与归一化网络同等性能
- 验证了归一化层的核心作用是管理训练初期的梯度问题

## 与综述的关联
属于 **Control** 维度的早期工作。比 ReZero/SkipInit 更早（ICLR 2019），初始化方案更复杂但更完整。它建立了"深度感知初始化"的理论框架，为后续 DeepNorm 等工作奠定基础。

## 核心贡献
首次系统性地证明：通过正确的初始化（而非归一化），可以训练任意深度的残差网络。
