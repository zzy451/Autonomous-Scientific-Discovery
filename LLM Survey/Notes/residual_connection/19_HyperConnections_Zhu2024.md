# Hyper-Connections

**论文信息**: Zhu, D., et al. (2024)
**arXiv**: 2409.19606
**分类**: Width (残差流宽度扩展)

## 核心思想
Hyper-Connections 将残差流从单通道扩展为 n 个并行通道（即 n 份隐状态副本），并引入两类可学习连接矩阵来管理跨通道和跨深度的信息流。

## 数学公式
标准残差: x_{l+1} = x_l + F(x_l)
HC: 将输入 x 扩展为 n 份 {x^1, x^2, ..., x^n}，通过两类可学习矩阵进行混合：
- **Depth-connections**: 对各通道的输入/输出赋予可学习权重（广义化的残差连接）
- **Width-connections**: 同一层内 n 个隐向量之间的信息交换

## 关键机制
- 将残差流的"宽度"概念引入：不再是单条残差高速路，而是多车道高速公路
- Depth-connections 控制每层的跳跃连接强度
- Width-connections 允许层内的多通道特征交互
- 参数和计算开销极小（可忽略不计）

## 实验结果
- 在 LLM 预训练中（dense 和 MoE 模型）显著优于标准残差连接
- 在视觉任务上同样有效
- 计算开销几乎可忽略

## 局限性
- 可学习混合矩阵的谱范数可能超过 1，导致信号在深层网络中指数增长
- 打破了残差连接的恒等映射性质，导致大规模训练时不稳定
- 这一问题催生了后续 mHC 的改进工作

## 与综述的关联
属于 **Width** 维度的开创性工作。首次系统化地提出"残差流宽度"概念，将单通道残差连接扩展为多通道。是 mHC 和 go-mHC 的基础。

## 核心贡献
从"宽度"维度重新审视残差连接设计空间，提出多通道残差流架构。
