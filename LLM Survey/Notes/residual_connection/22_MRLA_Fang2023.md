# Cross-Layer Retrospective Retrieving via Layer Attention (MRLA)

**论文信息**: Fang, Y., et al. (2023)
**arXiv**: 2302.03985 | **会议**: ICLR 2023
**分类**: Depth (跨层注意力检索)

## 核心思想
提出 Multi-Head Recurrent Layer Attention (MRLA)，一种跨层注意力机制。当前层的表示作为 query，对所有前序层的输出进行注意力检索，动态获取不同抽象层级的信息。

## 数学公式
MRLA: query 来自当前层 l，key/value 来自所有前序层 {1, ..., l-1}
Q_l = W_q * x_l
K_{1:l-1}, V_{1:l-1} = concat(x_1, ..., x_{l-1})
output_l = Attention(Q_l, K_{1:l-1}, V_{1:l-1})

## 关键机制
- **跨层回顾**: 当前层能够主动检索之前所有层的输出
- **多头机制**: 使用多头注意力捕获不同层之间的多种关系
- **递归结构**: 注意力计算是递归的（每层都聚合前序层）
- **轻量级版本**: 为降低计算成本，提出 lightweight MRLA

## 与其他方法对比
| 方法 | 跨层方式 | 权重类型 | 计算开销 |
|------|----------|----------|----------|
| 标准残差 | 仅相邻层 | 固定(+1) | 无 |
| DenseFormer | 所有前层加权平均 | 可学习标量 | 低 |
| MRLA | 注意力检索前层 | 输入依赖 | 中 |

## 实验结果
- 在视觉网络（CNN 和 ViT）上提升了表示能力
- 主要在视觉任务上验证，不是 LLM 领域
- 计算开销通过 lightweight 版本控制在合理范围

## 与综述的关联
属于 **Depth** 维度。是"跨层注意力"方法的早期代表。虽然主要在视觉领域验证，但其思想启发了后续 LLM 中的 DCA 和 AttnRes 等工作。

## 核心贡献
将跨层信息传递形式化为注意力检索问题，允许当前层动态选择性地利用前序层信息。
