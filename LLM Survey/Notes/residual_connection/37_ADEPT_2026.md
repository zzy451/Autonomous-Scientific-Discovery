# ADEPT: Adaptive Dynamic Early-Exit Process for Transformers

**论文信息**: Adaptive Dynamic Early-Exit Process for Transformers (2026)
**arXiv**: 2601.03700
**分类**: Dynamic Depth (动态深度 / Token 级早退)

## 核心思想
不同 token 的"难度"不同，简单 token 不需要经过所有层。ADEPT 提出自适应的 token 级早退机制，根据 token 复杂度动态调整计算量。关键创新在于解决了早退方法中跳过层的 KV cache 生成瓶颈问题。

## 关键机制
- **Token 级早退**: 每个 token 独立决定在哪一层退出，而非整个序列统一退出
- **解耦 KV Cache 生成**: 提出非顺序的解耦机制处理被跳过层的 KV cache，允许并行化计算
- **动态复杂度评估**: 根据 token 的中间表征动态判断是否需要继续深层计算
- **同时优化 prefill 和 generation**: 在预填充和生成两个阶段都实现加速

## 实验结果
- 语言生成任务上实现高达 25% 的效率提升
- 在不损失性能的前提下显著减少计算量
- KV cache 解耦机制解决了早退方法的实际部署瓶颈

## 与综述的关联
属于 **Dynamic Depth** 维度的最新工作。与 Mixture of Depths (Note 13)、DepthAdaptive (Note 14) 一脉相承，但更聚焦于 LLM 推理场景的实际部署问题。KV cache 解耦是对早退方法工程化的重要贡献。

## 核心贡献
提出解耦式 token 级早退框架，解决了跳过层 KV cache 生成的瓶颈，使动态深度方法在 LLM 推理中真正可用。
