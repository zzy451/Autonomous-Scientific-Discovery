# N-36: Muon 优化器与无归一化训练的交互 — 谱控制替代归一化 (2025-2026)

> **综合笔记**: 基于多篇论文的方向性总结
> **核心论文**:
>   - Muon: An optimizer for hidden layers (Keller Jordan et al. 2024)

>   - Muon is Scalable for LLM Training (arXiv: 2502.16982, 2025)
>   - Analyzing the Inexact Muon Update (arXiv: 2510.19933, 2025)
> **关键词**: Muon, 谱归一化, 正交更新, 无归一化, μP, Stiefel 流形, 优化器-归一化耦合

## 核心思想

Muon 优化器通过**正交化动量更新 (orthogonalized momentum updates)** 将权重矩阵的更新约束在 Stiefel 流形上，隐式地维持了权重矩阵的谱条件。这一特性与归一化层的功能产生了深刻的交互：如果优化器本身能维持权重的谱性质，那么归一化层的部分功能就变得冗余。这开辟了"**通过优化器设计替代归一化**"的新方向。

## Muon 优化器简介

### 核心机制

```
标准 Adam:
  m_t = β₁ m_{t-1} + (1-β₁) g_t        (一阶动量)
  v_t = β₂ v_{t-1} + (1-β₂) g_t²       (二阶动量)
  θ_t = θ_{t-1} - lr · m_t / (√v_t + ε)  (逐元素缩放)

Muon:
  m_t = β m_{t-1} + g_t                  (动量累积)
  U, Σ, V = SVD(m_t)                     (奇异值分解)
  θ_t = θ_{t-1} - lr · U V^T             (正交更新)
```

- Muon 的更新方向 $UV^T$ 是一个正交矩阵
- 这意味着每次更新都在 Stiefel 流形上进行
- 更新不改变权重矩阵的谱范数（所有奇异值的更新幅度相同）

### 与归一化的关系

| 归一化层的功能 | Muon 是否提供等价功能 |
|--------------|---------------------|
| 控制激活范数 | 间接提供（权重谱控制 → 激活范数受控） |
| 防止梯度爆炸/消失 | 提供（正交更新保持梯度范数） |
| 隐式学习率调节 | 提供（所有方向等幅更新） |
| 均值中心化 | 不提供 |
| 方差归一化 | 不直接提供 |

## 谱控制替代归一化 (arXiv: 2601.01306)

### 关键洞察

论文 "Ensuring Spectral Conditions throughout Training" 提出：
- 对于中等以上规模的模型，**仅在优化器更新层面维持谱控制**就足以保持 μP 兼容的缩放
- 这意味着可以**消除显式的归一化层**，同时保持超参数的跨规模迁移性
- 核心条件：优化器更新的谱范数需要被控制在 $O(1/\sqrt{d})$ 量级

### 与 Derf/DyT 的互补

| 方法 | 替代归一化的途径 | 作用位置 |
|------|----------------|---------|
| DyT (N-16) | 有界激活函数替代 | 前向传播（激活值） |
| Derf (N-17) | 更优的有界函数 | 前向传播（激活值） |
| nGPT (N-25) | 全面超球面约束 | 前向传播 + 权重 |
| **Muon + 谱控制** | **优化器正交更新** | **反向传播（权重更新）** |

这形成了一个有趣的对偶：
- DyT/Derf: 在**前向传播**中用有界函数替代归一化
- Muon: 在**反向传播**中用谱控制替代归一化的隐式效果

## Muon 的规模化 (arXiv: 2502.16982)

将 Muon 扩展到大规模 LLM 训练的两个关键技术：
1. **添加权重衰减**: 防止权重范数无限增长（归一化层原本隐式提供此功能）
2. **调整逐参数更新尺度**: 确保不同层的更新幅度与模型宽度正确缩放

这两个技术本质上是在**补偿移除归一化后丢失的功能**。

## 开放问题

1. **Muon + DyT/Derf 的组合**: 如果同时在前向（DyT/Derf）和反向（Muon）替代归一化，是否能完全消除归一化？
2. **计算开销**: Muon 的 SVD 计算比 Adam 昂贵，是否值得用来换取移除归一化？
3. **超大规模验证**: Muon 在百亿级模型上的表现仍需更多验证
4. **与 FP8 的交互**: 正交更新在低精度下的数值稳定性如何？

## 在综述中的定位

Muon 代表了"**从优化器侧替代归一化**"的新范式，与 DyT/Derf 的"从激活函数侧替代"形成互补。谱控制的思想将归一化的功能从"网络架构组件"转移到"优化算法属性"，暗示归一化的本质可能不在于特定的网络层，而在于维持训练过程中的某些不变量（谱条件、范数约束等）。这一视角为"归一化的本质是什么"这一根本问题提供了新的思考角度。
