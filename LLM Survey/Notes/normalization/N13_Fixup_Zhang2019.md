# Fixup Initialization: Residual Learning Without Normalization
**作者**: Hongyi Zhang, Yann N. Dauphin, Tengyu Ma | **年份**: 2019 | **arxiv**: 1901.09321

## 0. 摘要翻译

Batch Normalization 被认为是成功训练深层残差网络的关键组件。本文分析了深层残差网络中的梯度爆炸问题，发现其根源在于残差分支的方差随深度累积增长。我们提出 **Fixup (Fixed-update) Initialization**，一种精心设计的初始化方案，使得深层残差网络可以在**完全不使用归一化层**的情况下成功训练，并匹配使用 BatchNorm 的网络性能。Fixup 还消除了对学习率预热 (warmup) 的依赖。

## 1. 方法动机

### a) 为什么
BatchNorm 虽然有效，但带来了额外的计算开销、batch 依赖性和训练-推理不一致性。如果能通过正确的初始化完全替代 BN，将大幅简化网络设计。

### b) 痛点
- **BN 的固有问题**: batch 依赖、小 batch 不稳定、分布式训练同步开销
- **无 BN 时训练崩溃**: 标准初始化 (Xavier/He) 下，深层残差网络在去掉 BN 后无法训练
- **方差累积爆炸**: 残差连接使得每个 block 的输出方差递增，$L$ 层后方差 $\sim O(L)$
- **梯度爆炸**: 方差爆炸导致前向和反向传播中的数值不稳定

### c) 假设
BN 的主要功能是**在每一层控制激活值的尺度**。如果通过初始化直接确保每层的输出方差不会累积增长，就可以不需要 BN。关键是将残差分支最后一层的贡献缩放到接近零。

## 2. 方法设计

### a) Pipeline
1. 分析残差网络中的方差增长行为
2. 设计初始化方案使残差分支初始贡献足够小
3. 添加可学习标量偏置以补偿表达力
4. 完全移除所有 BN 层

### b) 模块

**方差分析**: 对于 $L$ 个残差 block，每个 block 有 $m$ 层，标准初始化下：
- 第 $l$ 个 block 输出: $x_l = x_{l-1} + f_l(x_{l-1})$
- 方差增长: $\text{Var}(x_L) \approx L \cdot \text{Var}(f_l)$（线性增长）

**Fixup 初始化方案**:
1. 残差分支中**最后一层**权重初始化为 **零**
2. 残差分支中**其他层**权重按 $L^{-\frac{1}{2m-2}}$ 缩放
3. 在每个残差分支中添加可学习**标量偏置** (scalar bias)

### c) 公式
- 缩放因子: 对每个残差分支中第 $i$ 层（$i < m$，即非最后一层）的初始化权重按 $L^{-\frac{1}{2m-2}}$ 缩放
- 最后一层初始化: $W_m = \mathbf{0}$（零矩阵）
- 可学习偏置: 在乘法前后添加标量偏置 $b_1, b_2$
- 效果: 初始化时残差分支输出为零，每个 block 等效于恒等映射
- 该策略保证: $\text{Var}(x_L) = O(1)$（方差不随深度增长）

## 3. 对比

| 特性 | 标准 ResNet + BN | Fixup (无 BN) |
|------|-----------------|--------------|
| 归一化层 | 每个 block 有 BN | 无 |
| 初始化 | He/Xavier | Fixup 初始化 |
| batch 依赖 | 有 | 无 |
| warmup 需求 | 通常需要 | 不需要 |
| 训练-推理一致性 | 不一致 | 一致 |
| 额外参数 | BN 参数 | 标量偏置 |
| 最大训练深度 | ~100 层 | ~100 层 |

## 4. 实验

- **CIFAR-10/100**: 110 层和 164 层 ResNet 无 BN，匹配有 BN 版本
- **ImageNet**: ResNet-50 无 BN，匹配有 BN 版本的 top-1 精度
- **机器翻译** (WMT14 En-De): 6 层 Transformer 无 LN，匹配有 LN 版本的 BLEU
- **消融实验**: 零初始化最后一层是最关键的设计选择
- **训练曲线**: 收敛速度与 BN 版本相当，无需 warmup

## 5. 总结

### a) 一句话
Fixup 通过将残差分支最后一层初始化为零、其他层按深度缩放，实现了无归一化层的深层残差网络训练，证明 BN 的核心功能可以通过初始化策略替代，直接启发了 T-Fixup、DyT 等后续工作。

### b) 速记 pipeline
```
残差分支最后一层 → 零初始化
残差分支其他层 → 按 L^(-1/(2m-2)) 缩放
添加标量偏置 → 移除所有 BN → 成功训练
（关键: 初始化时每个 block = 恒等函数）
```
