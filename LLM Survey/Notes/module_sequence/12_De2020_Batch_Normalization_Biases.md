# Batch Normalization Biases Residual Blocks Towards the Identity Function in Deep Networks
**作者**: Soham De, Samuel L. Smith | **年份**: 2020 (NeurIPS 2020) | **arxiv**: 2002.10444

## 0. 摘要翻译
Batch Normalization (BN) 极大地增加了残差网络的最大可训练深度，这一优势对深层残差网络在各类基准上的成功至关重要。然而，BN 为何能实现这一点仍不清楚。本文表明，BN 在初始化时将残差块偏置为近似恒等函数，随着网络深度增加，残差分支相对于跳跃连接被逐渐压制。基于此洞察，提出 SkipInit——一种简单的初始化方案，在每个残差分支末端引入可学习标量（初始化为零），无需任何归一化层即可训练极深的残差网络。

## 1. 方法动机
- BN 是训练深层残差网络的关键组件，但其真正机制长期不明
- 传统解释（减少内部协变量偏移、平滑损失面等）不够精确
- 作者发现 BN 的核心作用是在初始化时**缩小残差分支的输出幅度**
- 残差分支被 BN 缩小的因子约为 $1/\sqrt{L}$（$L$ 为网络深度）
- 这使残差块在初始化时近似恒等函数，简化了优化景观
- 问题：能否不用 BN，直接实现这种"恒等偏置"效果？

## 2. 方法设计
- **核心发现**: BN 的隐式效应 = 在初始化时将残差分支输出缩放为 $\sim 1/\sqrt{L}$
  - 残差块: $x_{l+1} = x_l + F_l(x_l)$
  - 有 BN 时: $\|F_l(x_l)\| \ll \|x_l\|$，因此 $x_{l+1} \approx x_l$
- **SkipInit**: 显式实现恒等偏置
  - 公式: $x_{l+1} = x_l + \alpha_l \cdot F_l(x_l)$
  - $\alpha_l$ 初始化为 0（或极小值）
  - $\alpha_l$ 为可学习参数，训练中逐渐增大
  - 实现仅需一行代码: `out = x + alpha * block(x)`
- **无需归一化层**: SkipInit 完全替代 BN 的可训练性功能

## 3. 对比
| 方法 | 机制 | 是否需要归一化 | 初始化行为 |
|------|------|--------------|-----------|
| 标准 ResNet + BN | BN 隐式缩小残差分支 | 需要 BN | 残差块 ≈ 恒等 |
| Fixup (Zhang 2019) | 精心设计的初始化方案 | 不需要 | 残差块 ≈ 恒等 |
| ReZero (Bachlechner 2020) | 可学习标量，初始化为 0 | 不需要 | 残差块 = 恒等 |
| SkipInit (本文) | 可学习标量，初始化为 0 | 不需要 | 残差块 = 恒等 |

- SkipInit 与 ReZero 思想相似，但本文的贡献在于**理论解释**：揭示 BN 的真正机制

## 4. 实验
- **架构**: ResNet-110, ResNet-1202 等深层 CNN
- **数据集**: CIFAR-10, CIFAR-100, ImageNet
- **结果**:
  - SkipInit 在无 BN 情况下达到接近 BN 的训练效果
  - 在极深网络 (1202层) 上，SkipInit 成功训练而无 BN 的标准初始化失败
  - BN 的学习率优势主要在大 batch 场景下显著，小 batch 下可忽略
  - 验证了"恒等初始化"是深层网络可训练性的充分条件
- **分析**: 追踪训练过程中 $\alpha$ 的演化，发现其从 0 逐渐增长，符合"先学恒等再学变换"的直觉

## 5. 总结
a) 一句话: BN 的真正作用是在初始化时将残差块偏置为恒等函数，SkipInit 用一个零初始化的可学习标量显式实现这一效果，无需归一化即可训练极深网络。
b) 速记 pipeline: `分析 BN → 发现隐式缩放 ~1/√L → 提出 SkipInit: x + α·F(x), α=0 → 无 BN 训练深层 ResNet`; 核心洞察: 归一化≠关键，恒等偏置=关键。
