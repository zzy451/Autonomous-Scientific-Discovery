# Fixup Initialization: Residual Learning Without Normalization
**作者**: Hongyi Zhang, Yann N. Dauphin, Tengyu Ma | **年份**: 2019 | **arxiv**: 1901.09321 | **发表**: ICLR 2019

## 0. 摘要翻译
Fixup（Fixed-update Initialization）提出了一种针对残差网络的结构化初始化方法，使得深层残差网络可以**完全不使用归一化层**进行训练。通过根据网络深度重新缩放残差分支的权重初始化，Fixup 防止了训练初期的梯度爆炸/消失问题。使用 Fixup 可以稳定训练 10,000 层的残差网络，并在图像分类和机器翻译（Transformer）任务上达到与使用归一化层的模型可比的性能。

## 1. 方法动机
- Batch Normalization（BN）被广泛认为是训练深层网络的必要组件
- 但 BN 引入了额外的计算开销、batch 大小依赖、以及序列模型不适用等问题
- 需要理解 BN 究竟提供了什么，以及是否可以用更简单的方法替代
- 挑战"归一化是必须的"这一流行认知

## 2. 方法设计（重点：模块排列顺序及其理论分析）

### Fixup 初始化策略
对残差网络的初始化进行以下修改：

#### 1) 残差分支末端初始化为零
- 将每个残差分支的最后一层权重初始化为 **0**
- 训练开始时，网络表现为恒等映射

#### 2) 深度相关的缩放
- 残差分支中其他层使用标准初始化（如 Kaiming 初始化）
- 乘以深度相关的缩放因子：`L^{-1/(2m-2)}`
  - L = 网络中残差块的总数
  - m = 每个残差块中的层数
- 确保前向传播中信号量级不随深度膨胀

#### 3) 可学习的标量参数
- 在每个残差分支添加可学习的标量乘法器（初始化为 1）
- 添加可学习的标量偏置（初始化为 0）
- 提供训练过程中的灵活性

### 理论保证
- 初始化时的残差为零 -> 输出等于输入
- 前向传播的方差被严格控制
- 梯度在反向传播中保持稳定
- 不需要任何归一化层来维持训练稳定性

### 与 Transformer 的关系
```
标准 Transformer: x + LN(SubLayer(x))   或   LN(x + SubLayer(x))
Fixup Transformer: x + 0 * SubLayer(x) -> x + alpha * SubLayer(x)  (alpha 学习)
```
- Fixup 在 Transformer 中的应用类似于 ReZero
- 证明了 LayerNorm 的核心作用是控制残差分支的初始量级

## 3. 与其他方法对比
| 方法 | 初始化策略 | 需要归一化 | 可训练深度 | 年份 |
|------|-----------|----------|-----------|------|
| 标准初始化 + BN | Kaiming/Xavier | 是 | ~100 层 | - |
| Fixup | 深度缩放 + 零初始化末端 | 否 | 10,000 层 | 2019 |
| ReZero | alpha=0 | 否 | 120 层 | 2020 |
| SkipInit | alpha=0 | 否 | 1,000 层 | 2020 |

## 4. 实验表现
- **CIFAR-10**：无 BN 的 ResNet 达到与 BN 版本可比的性能
- **ImageNet**：配合 Mixup 正则化，无 BN 训练达到 SOTA
- **10,000 层 ResNet**：成功稳定训练
- **Transformer 机器翻译**：在 WMT 基准上达到与使用 LayerNorm 的模型可比的 BLEU
- 验证了"归一化的核心作用是控制初始化量级"的假说

## 5. 总结
a) **一句话概括**：Fixup 通过深度相关的权重缩放和残差分支末端零初始化，证明了深层网络（包括 Transformer）可以完全不使用归一化层而稳定训练。

b) **速记 pipeline**：`x + L^{-1/(2m-2)} * SubLayer(x)，最后一层 init=0，加可学习 scalar`
