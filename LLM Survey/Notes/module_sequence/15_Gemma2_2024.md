# Gemma 2: Improving Open Language Models at a Practical Size

## 基本信息
- **作者**: Google DeepMind (Gemma Team)
- **年份**: 2024
- **arXiv**: 2408.00118
- **关键词**: Pre+Post Norm, RMSNorm, 实用规模LLM

## 核心贡献（Module Sequence 维度）
Gemma 2 采用了**前后双重 RMSNorm** 的归一化策略。

### 1. 归一化策略
- 对每个子层（Attention 和 FFN）的输入和输出**同时**施加 RMSNorm
- 相当于"三明治归一化"：`RMSNorm → Sublayer → RMSNorm`（加上残差连接）
- 与 CogView 的 Sandwich-LN 理念类似

### 2. 设计动机
- 标准 Pre-Norm（如 LLaMA 3 使用的）可能在深层训练时不够稳定
- 双重归一化可以更好地控制激活值范围
- 提供比纯 Pre-Norm 更强的训练稳定性

### 3. 其他架构特点
- Grouped Query Attention (GQA)
- RoPE 位置编码
- GeGLU 激活函数
- 知识蒸馏训练

### 4. 行业影响
- 作为 Google 的实际产品模型，其架构选择具有很强的参考价值
- 2B、9B、27B 三种规模验证
- 证明了 Sandwich-style Norm 在工业级 LLM 中的可行性

## 与综述的关联
- 是 Sandwich-LN / 双重归一化在工业 LLM 中的**实际应用案例**
- 与纯 Pre-Norm 路线（LLaMA 系列）形成对照
- 说明 Module Sequence 的选择在工业界仍是活跃的研究方向
- Pre+Post Norm 可能成为未来的趋势
