# Eagle and Finch: RWKV with Matrix-Valued States and Dynamic Recurrence
**作者**: Peng et al. | **年份**: 2024 | **arxiv**: 2404.05892

## 0. 摘要翻译
本文介绍 RWKV 系列的最新迭代 **RWKV-5 (Eagle)** 和 **RWKV-6 (Finch)**，对 RWKV-4 进行了重要改进。核心改进包括：(1) 引入 **矩阵值状态 (matrix-valued states)**，将隐状态从向量扩展为矩阵，显著提升信息容量；(2) RWKV-6 进一步引入 **动态递推 (dynamic recurrence / data-dependent decay)**，使衰减因子依赖于输入内容（类似 Mamba 的选择性机制）。这些改进使 RWKV 在保持线性复杂度的同时，在多项基准上达到或超过同规模 Transformer 的性能。模型从 0.4B 到 7.5B 参数不等。

## 1. 方法动机
a) **为什么提出这个方法**：RWKV-4 虽然实现了线性复杂度的序列建模，但其隐状态为向量形式，信息容量有限；且衰减因子（time decay）是固定的，缺乏 Mamba 的选择性。需要扩展隐状态和引入动态衰减。

b) **现有方法的痛点**：
- RWKV-4 的隐状态是向量（d 维），信息容量有限
- RWKV-4 的 time decay 是固定参数，不随输入变化（类似 RetNet 的固定 γ）
- 与 Mamba 相比，RWKV-4 缺乏输入依赖的选择机制

c) **研究假设/直觉**：
- 矩阵值隐状态（d × d_v）可以存储更多的 key-value 关联信息，类似于 Linear Attention 的外积展开
- 动态衰减让模型学会根据内容决定"记住多少旧信息"——关键信息的衰减慢，不重要信息的衰减快

## 2. 方法设计
a) **方法流程 (pipeline)**：

**RWKV-5 (Eagle)**：
1. 输入 x_t 生成 r_t (receptance/query), k_t (key), v_t (value)
2. 隐状态更新：S_t = diag(w) · S_{t-1} + k_t ⊗ v_t（矩阵值状态，w 为固定 decay 向量）
3. 输出：o_t = r_t · S_t
4. S_t 的维度为 d_k × d_v（矩阵），相比 RWKV-4 的向量大幅提升

**RWKV-6 (Finch)**：
1. 在 RWKV-5 基础上，decay 变为输入依赖：w_t = f(x_t)
2. 隐状态更新：S_t = diag(w_t) · S_{t-1} + k_t ⊗ v_t
3. w_t 由输入 x_t 经过线性变换和激活函数动态生成

b) **核心模块功能**：
- **矩阵值隐状态**：S ∈ R^{d_k × d_v}，存储所有 key-value 对的外积和（加衰减），信息容量 = d_k × d_v
- **Token Shift**：类似于 RWKV-4 的 time-mixing，对当前和前一步的 token 表示做线性插值作为 r, k, v 的输入
- **固定衰减 (RWKV-5)**：w 是可学习的固定向量，每个维度独立衰减
- **动态衰减 (RWKV-6)**：w_t = σ(x_t W_w + b_w)，衰减率随输入内容动态变化
- **多头结构**：多个隐状态头并行运行

c) **关键公式解释**：
- RWKV-5：S_t = diag(w) · S_{t-1} + k_t v_t^T，o_t = σ(r_t) · (S_t · 1)（简化表示）
- RWKV-6：S_t = diag(w_t) · S_{t-1} + k_t v_t^T，w_t = exp(-exp(f(x_t)))
- 信息容量：RWKV-4 = d（向量）→ RWKV-5/6 = d_k × d_v（矩阵）
- 训练时可展开为矩阵乘法形式实现并行

## 3. 与其他方法对比
- **与 RWKV-4 的区别**：
  1. 向量隐状态 → 矩阵隐状态（信息容量大幅提升）
  2. 固定衰减（RWKV-5）→ 动态衰减（RWKV-6）
- **与 Mamba 的比较**：
  - 两者都是线性递推模型，RWKV-6 的动态衰减类似 Mamba 的选择性 Δ
  - Mamba 的 SSM 参数 B, C, Δ 都输入依赖，RWKV-6 主要是 decay 输入依赖
  - Mamba 基于 SSM 理论，RWKV 基于 AFT/线性注意力传统
- **与 RetNet 的比较**：RetNet 也有指数衰减，但 RetNet 的 γ 是固定标量，RWKV-6 的衰减是向量且输入依赖
- **与 Linear Attention 的比较**：Linear Attention = 无衰减的矩阵值状态 RNN，RWKV-5/6 增加了衰减控制
- **创新点**：矩阵值隐状态 + 动态衰减的组合
- **适用场景**：长序列建模、流式推理、多语言 LLM

## 4. 实验表现
- **关键结果**：
  - RWKV-6 (Finch) 7B 在多项基准上接近或匹配 LLaMA-2 7B
  - 动态衰减（RWKV-6）相比固定衰减（RWKV-5）一致提升约 0.5-1.5 个百分点
  - 矩阵值状态相比向量状态提升显著
  - 多语言基准上表现优异（训练数据包含大量非英语语料）
  - 推理内存恒定（不随上下文长度增长）
- **优势场景**：多语言建模、流式推理、长上下文生成
- **局限性**：
  - 矩阵值隐状态增加了推理时的计算量（d_k × d_v 的矩阵乘法）
  - 在精确回忆（exact recall）任务上仍可能弱于全注意力
  - 模型规模验证到 7.5B，更大规模的验证有待补充
  - 社区生态和工具链不如 Transformer 成熟

## 5. 总结
a) **一句话概括（≤20字）**：矩阵隐状态 + 动态衰减，RWKV 对标 Mamba 选择性。

b) **速记 pipeline**：
`RWKV-5: S_t = diag(w)·S_{t-1} + k_t⊗v_t, o_t = r_t·S_t (矩阵状态，固定衰减)`
`RWKV-6: S_t = diag(w_t)·S_{t-1} + k_t⊗v_t (动态衰减: w_t=f(x_t))`
