# More Expressive Attention with Negative Weights (Cog Attention)
**作者**: Lv et al. | **年份**: 2024 | **arxiv**: 2411.07176

## 0. 摘要翻译
传统的 Softmax 注意力权重被限制为非负值，这限制了注意力机制的表达能力。本文提出 **Cog Attention**，通过允许注意力权重取负值来增强注意力的表达能力。核心思想是在标准 Softmax 注意力的输出中减去一个可学习的 **基准向量 (baseline vector)**，使得最终的等效注意力权重可以为负。这种设计使模型能够主动"抑制"不相关的上下文信息。实验表明，Cog Attention 在语言建模和各种下游任务上优于标准注意力，且增加的计算开销极小。

## 1. 方法动机
a) **为什么提出这个方法**：从理论角度，注意力权重受限于非负（Softmax 输出始终 >=0）意味着模型只能做 value 向量的凸组合，无法表达"排斥"或"抑制"特定信息的语义。

b) **现有方法的痛点**：
- Softmax 注意力只能对 value 做正向加权组合，无法表达负相关
- 在信息检索任务中，模型不仅需要"关注什么"，还需要"忽略什么"——标准注意力缺乏后者的直接机制
- 注意力矩阵的表达空间被限制在非负锥中

c) **研究假设/直觉**：如果允许注意力权重为负值，等价于扩大了注意力矩阵的表达空间，模型可以学会主动"减去"无关信息，就像认知过程中的选择性忽略。

## 2. 方法设计
a) **方法流程 (pipeline)**：
1. 计算标准的 Softmax 注意力输出：o = softmax(QK^T/√d) V
2. 学习一个基准向量 b（通过可学习的线性投影生成）
3. 最终输出 = o - b，即从注意力输出中减去基准向量
4. 等效地，注意力权重变为：a_i - 1/n（某些位置可以为负）

b) **核心模块功能**：
- **标准 Softmax 注意力**：正常计算注意力输出 o = Σ_i α_i v_i
- **基准向量 (Baseline)**：b = W_b · x（或所有 value 的加权均值），代表"默认/中性"信息
- **差分输出**：output = o - b = Σ_i α_i v_i - b = Σ_i (α_i - β_i) v_i，其中 β_i 是基准权重
- 当 α_i < β_i 时，等效注意力权重为负，实现对该位置的"抑制"

c) **关键公式解释**：
- CogAttn(Q,K,V) = softmax(QK^T/√d) V - Baseline(V)
- Baseline(V) 可以取为所有 V 的均值：b = (1/n) Σ_i v_i
- 等效注意力权重：w_eff_i = α_i - 1/n（可以为负）
- 或用可学习投影：b = sigmoid(q · w_b) · mean(V)

## 3. 与其他方法对比
- **与 DIFF Transformer 的比较**：DIFF Transformer 用两组 Softmax 做差分，CogAttn 用单组 Softmax 减去基准向量，CogAttn 方法更简单轻量
- **与标准 MHA 的本质区别**：打破了注意力权重的非负约束，扩展了表达空间
- **与 Sigmoid Attention 的比较**：Sigmoid 也能产生不归一化的注意力，但未直接引入负权重的概念
- **创新点**：从"认知抑制"角度出发，用极简的"减去基准"操作扩展注意力的表达能力
- **适用场景**：任何标准 Transformer 架构，即插即用，额外开销几乎为零

## 4. 实验表现
- **关键结果**：
  - 语言建模困惑度稳定下降（相比标准 MHA 约 0.5-1.5% 提升）
  - 在信息检索和选择性注意力任务上提升更明显
  - 分析显示部分 head 确实学到了显著的负权重注意力模式
- **优势场景**：需要区分"关注"与"抑制"的场景（如文档检索、多选问答）
- **局限性**：
  - 在某些任务上提升幅度较小，不如 DIFF Transformer 的提升显著
  - 基准向量的设计有多种选择，最优设计可能因任务而异
  - 理论分析较多，大规模预训练验证相对有限

## 5. 总结
a) **一句话概括（≤20字）**：减去基准向量让注意力权重可为负，增强表达力。

b) **速记 pipeline**：
`Input → Q,K,V → Softmax Attn Output → 减去 Baseline(V) → 等效负权重注意力 → Output`
