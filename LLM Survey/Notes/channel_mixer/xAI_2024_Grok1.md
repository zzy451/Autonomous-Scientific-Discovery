# Grok-1: xAI Open-Source MoE Model

**论文**: Grok-1 Technical Release
**作者**: xAI
**年份**: 2024
**来源**: x.ai blog / GitHub
**标签**: MoE, 大规模LLM, 开源

---

## 0. 摘要
Grok-1 是 xAI 在 2024 年 3 月开源的 314B 参数 MoE 模型，使用 8 个专家中激活 2 个的 Top-2 路由策略。64 层 Transformer，6144 维嵌入，48 个查询头，8 个 KV 头。

## 1. 方法动机
a) xAI 致力于构建高性能、可访问的 AI 系统。
b) 大规模 MoE 可以在有限的推理成本下获得更大的模型容量。
c) 通过开源促进社区发展。

## 2. 方法设计
a) 架构配置：
   - 总参数 314B
   - 8 个专家，每 token 激活 2 个（约 25% 参数活跃）
   - 64 层 Transformer
   - 嵌入维度 6144
   - GQA: 48 查询头 + 8 KV 头

b) 其他特征：
   - RoPE 位置编码
   - 8192 最大上下文长度
   - SentencePiece 分词器（131072 词汇量）
   - 支持激活分片和 8-bit 量化

c) 技术栈：JAX + Rust

## 3. 与其他方法对比
| 方法 | 总参数 | 活跃参数 | 专家配置 |
|------|--------|----------|----------|
| Mixtral 8x7B | 47B | 13B | 8选2 |
| **Grok-1** | **314B** | **~78B** | **8选2** |
| DBRX | 132B | 36B | 16选4 |

## 4. 实验表现
- 开源的是未经微调的 base 模型检查点
- 预训练截止于 2023 年 10 月
- Apache 2.0 许可证

## 5. 对 Channel Mixer 的意义
Grok-1 是工业级 MoE LLM 的代表之一，展示了 Top-2 路由+8专家的大规模实践。其 314B 参数量证明了 MoE 架构在超大规模下的可行性。

## 6. 总结
a) 核心思想：314B参数8选2MoE开源大模型（14字）
b) 速记 pipeline：
   1. 64 层 Transformer + 8 个 FFN 专家
   2. Top-2 路由激活 ~25% 参数
   3. GQA 注意力 + RoPE 位置编码
   4. 开源权重，JAX + Rust 实现
