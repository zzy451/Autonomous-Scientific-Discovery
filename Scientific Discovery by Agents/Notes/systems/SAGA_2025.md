# SAGA 2025

## 基本信息

- **论文**: Accelerating Scientific Discovery with Autonomous Goal-evolving Agents
- **年份**: 2025
- **来源**: arXiv:2512.21782
- **系统名称**: SAGA
- **关键词**: autonomous goal-evolving agents, scientific discovery, bi-level architecture, objective evolution, scoring functions

## 一句话总结

SAGA 通过 goal-evolving agents 将科学发现中的目标函数和评分函数纳入演化对象：外层 agent 分析优化结果、提出新 objectives 并转化为 computable scoring functions，内层进行候选 solution optimization。

## 研究问题

许多 evolutionary discovery 系统只优化给定目标，例如提高某个 benchmark score 或实验指标。但科学发现中的难点常常是目标本身会变化：新结果会暴露新的约束、失败模式或更有意义的探索方向。SAGA 试图让目标也参与迭代演化。

## 系统架构

SAGA 可理解为 bi-level discovery architecture：

| 层级 | 作用 |
|---|---|
| Outer goal-evolving agents | 分析当前结果，提出新目标或修改目标 |
| Objective translation | 将自然语言目标转化为可计算 scoring functions |
| Inner optimization loop | 在给定目标下优化候选 solution |
| Evaluation feedback | 将结果反馈给外层 agents，触发目标更新 |

## XYZ 分类

暂定分类为：

| 维度 | 标注 | 理由 |
|---|---|---|
| X | `X4` | 外层目标演化 agent 与内层优化形成层级结构 |
| Y | `Y5` | 目标、评分函数和 discovery harness 本身被更新 |
| Z | `Z2,Z3,Z4,Z5,Z7,Z8` | 覆盖目标/问题形成、设计、执行、分析、验证和迭代 |

建议作为 **严格 ASD 候选**，但 X 轴需深读确认：若外层并非显式多角色 agents，则可能降为 `X2` 或 `X0`。

## 对综述的价值

SAGA 可以补 `X4-Y5-*`，并给第 5 章提供一个非常重要的概念：harness evolution 不只是更新 memory/tool/role，也可以更新 **scientific objective and scoring function**。

## 局限性

- X 轴分类依赖原文是否清楚展示多 agent 组织。
- 目标函数自动生成可能把科学重要性简化为可计算 proxy。
- 需要关注其验证是否超出 benchmark 或 toy discovery tasks。
