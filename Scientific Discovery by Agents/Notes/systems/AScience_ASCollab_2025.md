# AScience / ASCollab 2025

## 基本信息

- **论文**: Hypothesis Hunting with Evolving Networks of Autonomous Scientific Agents
- **年份**: 2025
- **来源**: arXiv:2510.08619
- **系统名称**: AScience / ASCollab
- **关键词**: autonomous scientific agents, evolving networks, hypothesis hunting, agent collaboration, peer review

## 一句话总结

AScience / ASCollab 将科学发现建模为 agents、networks 和 evaluation norms 的共同演化：多个 autonomous scientific agents 可以围绕问题形成协作网络、产生和检验假设，并通过类似 peer review 的机制评价发现。

## 研究问题

大多数多智能体科学系统仍是封闭团队：角色、工具和通信拓扑都由设计者预设。AScience / ASCollab 关注更开放的问题：科学发现是否可以由一组 autonomous agents 通过协作、竞争和评价机制自组织形成。

这使它非常适合本文中 `X5: 开放能力网络` 的讨论，尤其用于补充 Science Earth 之外的证据。

## 系统架构

从综述角度，AScience / ASCollab 的关键是 networked scientific agents：

| 层面 | 含义 |
|---|---|
| Autonomous scientific agents | 每个 agent 具有研究、假设生成或评价能力 |
| Evolving collaboration network | agent 之间的连接和协作关系可随任务变化 |
| Evaluation norms | 结果通过互评、比较或类似同行评审的机制接受评价 |
| Discovery trace | 假设、实验尝试和评价形成可追踪链条 |

## 对综述的价值

AScience / ASCollab 可以补 `X5-Y5-*` 坐标，尤其是 `Z4/Z5/Z8`。它把 open capability network 的证据从 Science Earth 的平台/协议视角，扩展到 evolving agent collaboration network。

## 局限性

- X5 口径需要谨慎：它是 evolving agent network，不一定等同 Science Earth 的 open capability market。
- 若任务主要是模拟或 benchmark，严格 ASD 身份需降级。
- peer review 式评价不能替代实验、形式证明或真实世界验证。
