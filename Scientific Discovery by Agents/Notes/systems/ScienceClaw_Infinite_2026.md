- **论文**: ScienceClaw + Infinite: A Decentralized Multi-Agent Ecosystem for Scientific Discovery
- **年份**: 2026
- **来源**: arXiv:2603.14312
- **系统名称**: ScienceClaw + Infinite
- **关键词**: decentralized agents, skill registry, artifact DAG, global index, discourse layer, scientific ecosystem

## 摘要要点

ScienceClaw + Infinite 提出一个 decentralized scientific agent ecosystem。系统包含 decentralized agents、300+ skill registry、artifact DAG、global index 和 discourse layer，使 agents 可以发布、检索、组合和讨论科学产物与能力。它是 `X5-Y5` 的强证据：多智能体不是封闭团队，而是开放的能力/产物网络。

## 方法动机

端到端单 agent 或固定团队难以覆盖科学发现所需的全部能力。科学研究需要工具、数据、技能、讨论、产物版本和 provenance 的持续交换。ScienceClaw + Infinite 的目标是构建一个开放生态，使 agents 能在 shared skill/artifact space 中协作和演化。

## 方法设计

系统包含以下层：

1. decentralized agents：不同 agents 独立运行并贡献能力或产物。
2. skill registry：登记可复用 scientific skills，规模超过 300 个。
3. artifact DAG：保存论文、代码、数据、实验计划、结果等产物及依赖。
4. global index：支持跨 agent 和跨产物检索。
5. discourse layer：允许 agents 围绕产物讨论、批评和改进。
6. provenance / version structure：使产物可追踪和复用。

这里演化的是开放能力网络和 artifact graph，而不是单个 hypothesis。

## 实验与结果

论文展示该生态能支撑科学技能发现、产物复用和跨 agent 协作。其贡献更偏 agentic science infrastructure，而不是单个领域发现 benchmark。

## 局限性

- 开放生态的质量控制、权限、安全和信任机制仍是挑战。
- 产物网络能促进协作，但不自动保证科学正确性。
- 需要更多真实科学任务证明其相对固定团队的优势。
