# TAIS 2024

## 基本信息
- **论文**: Toward a Team of AI-made Scientists for Scientific Discovery from Gene Expression Data
- **作者**: Haoyang Liu, Yijiang Li, Jinglin Jian, Yuxuan Cheng, Jianrong Lu, Shuyi Guo, Jinglei Zhu, Mianchen Zhang, Miantong Zhang, Haohan Wang
- **年份**: 2024
- **来源**: arXiv:2402.12391
- **关键词**: gene-expression, team-of-agents, disease-gene-discovery

## 核心思想
TAIS 使用模拟科研团队来自动化基因表达数据中的疾病预测基因发现。

## 方法细节
系统包含 project manager、data engineer 和 domain expert 等角色，由 LLM 扮演并协作完成数据选择、处理和分析。

## 关键结果
论文构建 benchmark 并展示系统在 gene identification 上的潜力。

## 局限性
仍是早期框架，严谨验证和泛化能力有限。

## 核心贡献
较早将“AI-made scientists team”用于基因表达科学发现。

## 与综述的关联
适合放入 `X2-Y1-Z1/Z3/Z4/Z5,Z7`，支撑固定角色多智能体科学数据分析。

## 原文核对与分类复核
- **原文核对**：原文模拟 project manager、data engineer、domain expert 等角色，面向 gene expression data 中 disease-predictive genes 识别。
- **机制判断**：它是多角色科学数据分析系统，支撑 ASD 的数据分析和发现环节，但不是完整自主科学发现闭环。
- **分类复核**：保持 `ASD 相关系统`；`X2` 正确；`Y1` 可保留为协作式分析/修正；`Z1,Z3,Z4,Z5,Z7` 正确。
- **写作用途**：适合作为 role specialization 在生物数据分析中的早期证据。

## 深读补充（按 MARS 标准）

### 研究问题
TAIS 关注 gene expression data 中 disease-predictive genes 识别需要大量人工数据处理、分析和领域解释的问题。

### 系统架构 / 工作流
系统模拟 project manager、data engineer、domain expert 等 AI-made scientists，以多角色协作方式复现数据科学团队的分析流程。

### 关键机制
核心是 role specialization。不同 agents 承担项目管理、数据处理和领域解释职责。

### 实验验证与证据
原文构建 benchmark dataset 评估 gene identification 效果，展示多角色 LLM 系统提高分析效率的潜力。

### 局限性补充
TAIS 是生物数据分析系统，严格科学发现仍需要实验验证和生物机制解释。

### XYZ 分类逐项解释
- `X2`：固定角色多智能体。
- `Y1`：协作式分析和修正。
- `Z1,Z3,Z4,Z5,Z7`：覆盖知识、分析设计、执行、结果和验证。

### 综述写作用法
适合多智能体科学劳动分工的早期例子。
