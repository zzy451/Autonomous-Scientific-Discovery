## 从单Agent到证据链：记忆系统的逐步演进

这10个记忆系统呈现出一条清晰的演进逻辑链，每一环解决前一环留下的未竟问题：

**第一环：单Agent上下文扩展。** MemGPT（2023）回答了最基础的问题——LLM上下文窗口有限，如何在单个Agent内装下"无限"信息？它从操作系统虚拟内存借来分页思想，用主上下文（RAM）+外部存储（磁盘）双层架构，让LLM自主决定何时检索、写入和驱逐。这是记忆系统的"容量革命"。

**第二环：记忆不只是存，更要会用。** Generative Agents（2023）在MemGPT同一年的独立路线指出：存得更多不是目的，从记忆中提炼高层抽象才是。记忆流→检索→反思→规划四阶段将记忆从被动日志升级为主动知识加工系统——反思将日常观察合成抽象洞察，规划将抽象目标递归分解为可执行动作。这是记忆系统的"质量革命"。

**第三环：记忆不能是黑箱。** Memory Sandbox（2023）从前两个系统的自动化方向抽身，提出了一个根本性的人机交互问题：用户如果看不见Agent记住了什么，就无法信任它，也无法在记忆出错时干预。它将记忆"对象化"——在界面上可查看、编辑、删除、总结和跨对话共享。这是记忆系统的"透明性革命"，也为后来的记忆可审计性需求埋下伏笔。

**第四环：从单用户到多用户的权限建模。** Collaborative Memory（2025）直面真实场景：科研协作不是一个人和一个Agent对话，而是多个研究者在不同权限下共享知识。二分图访问控制G_UA(t)∪G_AR(t)将谁可以调用哪个Agent、哪个Agent能访问哪些资源形式化为随时间演化的数学结构，每条记忆携带不可变溯源。这是记忆系统的"安全协作革命"。

**第五环：多Agent间需要语义协议。** MMP（2025/26）进一步追问：即使权限问题解决了，多个Agent之间如何"互相理解"？它设计了协议层——CAT7七字段固定schema确保互操作性，SVAF让接收者按自身角色逐字段评估（而非全盘接受），DAG谱系O(1)回声检测防止信息循环，remix写入时过滤让跨会话重启时恢复的是经角色评估的理解。这是记忆系统的"协议化革命"。

**第六环：记忆应该自主组织与演化。** A-MEM（2025）回到单Agent内部，但提出了更根本的问题：记忆结构不应由开发者预定义。受Zettelkasten卡片盒启发，它让LLM自主生成结构化笔记、判断链接、并让新记忆触发旧记忆的上下文更新。这是记忆系统的"自主性革命"——从"人定义schema"到"Agent自己组织知识网络"。

**第七环：单一记忆类型不够，需要多类型组合。** MIRIX（2025）批判了前序工作的扁平化假设：人类记忆是多类型的（情景的、语义的、程序的……），为何Agent记忆只有一种？六种记忆类型（Core/Episodic/Semantic/Procedural/Resource/Knowledge Vault）×8个协调Agent提供了较完整的分类架构，并在多模态场景（20K截图）上验证了组合式记忆的优势。这是记忆系统从扁平存储走向类型分化的重要证据。

**第八环：每个角色的记忆应该不同。** Intrinsic Memory Agents（2025）将多类型记忆进一步推向角色维度——多Agent系统中理论家、实验家、数据分析师各应维护自己视角的专属记忆，从自身输出中内在演化而非依赖外部摘要。这解决了"记忆同质化消灭角色专业化"的深层矛盾。这是记忆系统的"异质性革命"。

**第九环：图结构是记忆的统一范式。** Graph Agent Memory（2026）站在以上所有工作的上方，提供了一个元视角：所有记忆形式——线性缓冲、向量存储、键值日志——都是图的退化特例。知识图谱、层级图、时序图、超图、混合图五种构造范式覆盖了从事实性知识到经验轨迹的全部记忆需求，四阶段生命周期（提取→存储→检索→演化）为记忆系统设计提供了统一方法论。这是记忆系统的"统一框架革命"。

**第十环：整合本身可能是问题所在。** Memory Warning（2026）以尖锐的否定性发现为整条演进链提供警戒：前九环都在追求更智能的记忆整合——更自动的链接、更频繁的更新、更抽象的提炼——但GPT-5.4在被证明能100%正确解决19个ARC-AGI问题后，经10轮流式整合准确率跌至52.6%。三种失败模式（错误分组、过度泛化、过拟合）直指整合步骤本身可能成为退化源头。这不是否定前九环，而是迫使整个领域重新校准设计哲学：原始情景痕迹应作为一等公民永久保留，整合应选择性、延迟、并始终锚定于可回溯的原始证据。这是记忆系统从"信任整合"转向"信任痕迹"的重要依据。

纵观这10个系统，演进方向清晰可辨：**单Agent容量扩展 → 记忆质量提炼 → 人机透明交互 → 多用户权限安全 → 多Agent语义协议 → 自主组织演化 → 多类型分化 → 角色异质记忆 → 图统一框架 → 情景-抽象分离的证据链原则**。每一步都在回答前一步的盲区，最终指向一个共同的目标：让Agent的记忆不仅"更多"，而且"更可靠、更透明、更安全、更自主、更结构化、更有迹可循"。
