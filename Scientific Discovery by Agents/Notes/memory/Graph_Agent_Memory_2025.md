# Graph Agent Memory 2025

## 基本信息
- **论文**: Graph-based Agent Memory: Taxonomy, Techniques, and Applications
- **作者**: Chang Yang, Chuang Zhou, Yilin Xiao, Su Dong, Luyao Zhuang, Yujing Zhang, Zhu Wang, Zijin Hong, Zheng Yuan, Zhishang Xiang, Shengyuan Chen, Huachi Zhou, Qinggang Zhang, Ninghao Liu, Jinsong Su, Xinrun Wang, Yi Chang, Xiao Huang
- **年份**: 2026
- **来源**: arXiv:2602.05665
- **关键词**: memory, graph-memory, taxonomy

## 核心思想

本文是一篇较全面的基于图的 LLM Agent 记忆系统综述，由香港理工大学、厦门大学、新加坡管理大学等联合完成，配套资源库 https://github.com/DEEP-PolyU/Awesome-GraphMemory。
文章指出传统线性/向量/键值存储范式在建模关系依赖、层级组织和因果链方面存在根本缺陷，图结构凭借实体-关系显式建模、层级语义捕获、灵活遍历推理三大能力成为 2025-2026 年 Agent 记忆研究的前沿。
核心贡献包括：三维分类法（时序维度×内容维度×结构维度）、四阶段生命周期框架（Extraction → Storage → Retrieval → Evolution）、以及涵盖知识图谱/层级图/时序图/超图/混合图共计五大图构造范式的系统技术综述。

## 研究问题与动机
LLM Agent 面临三大瓶颈：(i) 知识截止——预训练数据静态边界导致无法接入实时信息（如当前金融数据）或领域专有知识；(ii) 工具能力不足——现有 LLM 在学习与高效应用新工具方面能力受限；(iii) 性能饱和——在迭代式长周期任务中因无法积累任务特定洞察而反复犯同样错误。记忆系统被定位为解决这些问题的核心模块，承载四个关键目标：个性化定制（捕获用户偏好与交互历史，如记住编码工作流习惯）、长程推理超越上下文窗口（提供无界外部存储以支持跨时间尺度的自适应学习）、无参数更新的自我改进（通过积累经验知识与反馈实现性能迭代提升）、以及通过结构化可验证内容缓解幻觉（将输出锚定于显式可验证的记忆内容）。

然而现有 Agent 记忆系统缺乏统一的分类框架和系统性技术对比。"短期vs长期""知识vs经验"等传统二分法已不足以刻画当代图记忆系统的丰富度——2025-2026 年涌现的知识图谱、时序图、超图、层级图、混合图等多样化图结构各有其设计权衡，但缺乏统一的形式化框架来组织这些技术选项。构建者缺少从"记忆生命周期"视角进行的全链路设计指导：从提取、存储、检索到演化的每个阶段该如何选型和优化，不同阶段之间的技术依赖关系如何。

## 方法细节

**三维分类法**：时序维度区分短期记忆（当前上下文、活跃推理链、瞬态状态）与长期记忆（跨会话持久化知识）；内容维度区分知识记忆（被动、静态、客观事实，如通用规则、领域知识库）与经验记忆（主动、动态、个人交互日志）；结构维度区分非结构化（线性缓冲、向量存储、键值日志）与结构化图记忆。同时引入认知结构分类：语义记忆、程序记忆、联想记忆、工作记忆、情景记忆、情绪/情感记忆（Sentiment Memory）六类。

**四阶段生命周期框架**：(1) 记忆提取——从文本源（结构化信息抽取/语义嵌入/摘要）、时序轨迹（事件分割/状态快照/模式挖掘）、多模态数据（描述生成/交互内容检测/联合嵌入）三类原始数据中构建结构化记忆；(2) 记忆存储——五种图构造范式：知识图谱（三元组建模与LLM驱动的冲突消解）、层级图（语义聚类+递归摘要，有向无环图建模父子关系）、时序图（双时态建模区分有效时间与记录时间、事件时间与提及时间解耦、层次时间约束避免时序幻觉）、超图（保持N元关系完整性，避免二元分解导致的信息碎片化）、混合图（知识-经验解耦，如Optimus-1的HDKG+AMEP架构）；(3) 记忆检索——六种基础算子（相似度/规则/时序/图遍历/强化学习/Agent驱动）与三种增强策略（多轮检索、检索后处理、混合源检索）；(4) 记忆演化——内部分自我进化（整合合并/冲突消解/遗忘裁剪）与外部分自我探索（主动信息收集/环境反馈驱动更新）。

## 关键结果
1. 图结构在关系依赖性建模方面具有天然优势——传统线性记忆可视为图的退化特例（链状图），向量记忆可视为带相似度加权边的全连接图，即便是普通的平坦记忆也可视为无显式关系编码的退化图，因此图记忆提供了统一且可扩展的框架，而不仅仅是替代方案。
2. 记忆演化是当前最不成熟但最关键的生命周期阶段——Kimi K2.5、MATRIX、AgentEvolver等系统虽已开始探索外部自我探索，但内部自演化（整合抽象/冲突检测/遗忘）的可靠性和理论支撑严重不足。
3. 缺乏统一的评估基准是图记忆方法间公平比较的主要障碍——现有评测散落在不同领域（对话/游戏/科学发现），对图特有指标（关系保真度、拓扑一致性、时序正确性）覆盖不足。
4. 知识记忆与经验记忆的图结构选择存在清晰对应——知识记忆适配强调显式关系和包含结构的图（知识图谱），经验记忆适配强调时序序列和交互轨迹的图（时序图），混合/多层图在集成两者时取得最佳效果。
5. RL驱动和Agent驱动的检索策略代表了从"固定启发式"到"上下文自适应"的范式转变——Mem-α、Memory-R1、TGM等通过GRPO/PPO优化检索策略，使召回更贴合下游任务需求而非表面语义相似度。

## 局限性

- 作为综述性工作无原创实验验证，所有分析结论依赖已发表论文的可复现性——部分被引系统仅经过单次实验或特定 benchmark 验证，泛化性存疑。
- 四阶段生命周期框架的各阶段边界在工程实践中可能模糊：例如提取与演化常交织进行（增量更新时新事实的提取和旧事实的冲突消解同时发生），检索与演化也可能耦合（检索触发记忆强化或遗忘）。
- 对科学发现等特定垂直领域（科学假设生成、实验设计、因果推断）的深入讨论不足：缺乏面向跨实验可比性、假设版本管理、不确定性量化等科学场景特需的领域定制化图记忆架构建议。
- 超图、混合图等新兴结构虽理论优势清晰，但缺乏足够多的独立评估数据支撑——目前大部分评测仍集中在知识图谱和层级图上，外推风险高。
- 图结构构建与维护的计算开销缺乏定量分析：大规模实时更新（千万级节点）场景下的延迟预算、存储增长曲线、增量索引策略的经济性评估均未覆盖。

## 核心贡献
本文建立了图基 Agent 记忆的一份较完整技术地图，其生命周期视角的核心洞见在于：记忆不应只是被动的"日志"，还可以是主动的、结构化的"经验拓扑模型"——不仅记录"发生了什么"，更关键的是建模"这些事件如何连接"。传统记忆形式可视为此框架的特例（线性缓冲=链状图，向量存储=全连接带权图），因此图记忆提供了统一范式而非替代方案。综述者强调记忆演化、图质量、可扩展性/效率、隐私安全、可解释性、理论基础和多 Agent 记忆协调仍是关键挑战；配套 GitHub 仓库收录相关论文、开源数据和项目。

## 与综述的关联

该综述为本项目科学发现 Agent 记忆系统章节提供了核心分类框架和技术路线图。科学知识天然具有图结构——概念网络（化合物-反应-产物）、引用关系（论文-引用-论文）、因果链（条件-现象-机制）、实验-假设多层图（实验节点通过"支持/反驳"边连接假设节点），图记忆对此尤为适用。其生命周期视角中的演化阶段与科学知识迭代高度一致——科学家在整合新实验证据时不断更新内部知识图，这一过程直接对应记忆演化的内/外双循环机制。该文中多轮检索、图遍历检索、时序约束检索等技术与科学发现中"检索相关文献-建立概念连接-追踪实验时序"的需求高度匹配，可直接指导本综述记忆系统章节的技术工具选取和组织框架设计。
